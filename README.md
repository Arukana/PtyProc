# PtyProc
[![Build Status](https://travis-ci.org/Arukana/PtyProc.svg?style=flat-square&branch=master)](https://travis-ci.org/Arukana/PtyProc)

#### How to build
```shell
git clone https://github.com/Arukana/PtyProc.git PtyProc && cd PtyProc
cargo build
```
